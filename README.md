# Personal Portfolio Website

## Assignment Overview
This project is a personal portfolio website created as part of my Computer Fundamentals assignment. It demonstrates the use of HTML and CSS, as well as git and GitHub for version control.

## Features
- Home page introducing myself and my skills
- About page
- Contact information/form
- Profile photo

## Technology Used
- **HTML**
- **CSS**
- **Git** (for version control)
- **VS Code** (development environment)
- **GitHub** (repository hosting)

## How to View Locally
1. Clone the repository.
2. Open the folder and double-click `index.html` to open it in your web browser.

## Repository Contents
- `index.html` – Main portfolio home page
- `about.html` – About me section
- `projects.html` – Projects gallery
- `contact.html` – Contact form
- `photo.jpeg` – Profile image asset
- `README.md` – Project description and instructions

## Version Control Workflow
- Initialized repository with `git init`
- Used `git add` to track changes
- Committed updates with `git commit`
- Pushed all work to GitHub with `git push`
- Made repo public for instructor review
